# TECHNICAL IMPLEMENTATION DETAILS

## Translation Key Naming Convention
```
namespace:category.subcategory.key

Examples:
- admin:dashboard.businessIntelligence.title
- admin:dashboard.businessIntelligence.monthlyRevenue
- supervisor:dashboard.studentsNeedingAttention
- supervisor:dashboard.qualityTrend.stable
```

## Component Translation Pattern
```tsx
// Standard implementation for all components
import { useTranslation } from 'react-i18next';
import { useLanguage } from '@/hooks/useLanguage';

const ComponentName = () => {
  const { t } = useTranslation(['namespace1', 'common']);
  const { direction, isRTL } = useLanguage();

  return (
    <div className={cn("component", { "rtl": isRTL })}>
      <h1>{t('namespace1:section.title')}</h1>
      <p>{t('common:actions.save')}</p>
    </div>
  );
};
```
